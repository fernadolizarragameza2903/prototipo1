# 7. Codigo funcional inicial

## Que se debe mostrar

El codigo funcional inicial es la maqueta trabajando en el navegador. No necesita backend real ni inteligencia artificial real, pero si debe verse profesional y permitir interaccion.

## Funciones implementadas

- Login demo para estudiante.
- Navegacion entre paginas.
- Encuesta STEM de 5 preguntas.
- Resultados dinamicos con perfil STEM.
- Recomendaciones de carreras, videojuegos y videos.
- Chatbot IA simulado.
- Dashboard con XP, madurez e intereses.
- Mini proyectos filtrables.
- Panel docente simulado.
- Arquitectura, APIs, base de datos y seguridad simuladas.
- Guardado de progreso con `localStorage`.

## Como probarlo

1. Abrir `index.html`.
2. Ingresar con:
   - Usuario: `Fernando`
   - Contrasena: `12345`
3. Completar la encuesta.
4. Revisar los resultados.
5. Abrir `dashboard.html`.
6. Abrir `ia.html` y probar el chatbot.
7. Abrir `proyectos.html` y cambiar filtros.
8. Abrir `docente.html`.

## Que se evalua

- Funcionalidad visual.
- Interaccion del frontend.
- Coherencia UX/UI.
- Navegacion clara.
- Presentacion profesional.
