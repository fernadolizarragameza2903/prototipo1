# 8. Enlace GitHub

## Que se debe presentar

El enlace del repositorio publico donde se encuentra el codigo del proyecto.

## Formato recomendado

```text
https://github.com/usuario/gran-mente
```

Cuando el repositorio este creado, reemplazar `usuario` por el nombre real de la cuenta de GitHub.

## Que debe tener el repositorio

- Codigo completo del prototipo.
- README actualizado.
- Carpeta `docs` con explicaciones, diagramas o capturas.
- Commits descriptivos.
- Estructura clara de carpetas.
- Proyecto publicado en GitHub Pages si es posible.

## Commits recomendados

Ejemplos de mensajes:

```text
Inicializa prototipo GRAN MENTE
Agrega encuesta STEM y resultados dinamicos
Implementa dashboard y chatbot IA simulado
Agrega panel docente y arquitectura simulada
Actualiza README y documentacion del proyecto
```

## GitHub Pages

Si se publica en GitHub Pages, agregar tambien el enlace de despliegue:

```text
https://usuario.github.io/gran-mente/
```
